AuraCare - React integration

This repository was incrementally scaffolded with a minimal React + Vite setup so you can deploy or continue migrating pieces of the UI into React components.

Quick start:

1. Install dependencies

```powershell
npm install
```

2. Run development server

```powershell
npm run dev
```

Notes:
- The original static site files remain in place (`index.html`, `css/`, `js/`, `assets/`).
- A light React app mounts into `#react-root` without removing the existing content. Migrate pieces into `src/` as needed.
